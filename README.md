# react_app

## 手順
次の手順で実行
1. クローン
```sh
% git clone this repo
```
2. cd react_app/Docker
```sh
% cd react_app/Docker
```
3. 構築
```sh
% docker-compose build
```
4. 起動
```sh
% docker-compose up -d
```

5. access http://localhost:3000

<br>

## ログ
Dockerディレクトリで下記コマンド
```sh
% Docker logs -f react_app
```
